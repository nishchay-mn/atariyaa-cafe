# Atariyaa Cafe — अटरिया कैफ़े 🍵

A modern, Gen-Z aesthetic website for **Atariyaa Cafe**, Malviya Nagar, Jaipur.

🔗 **Live Site:** https://YOUR-USERNAME.github.io/atariyaa-cafe

---

## ✨ Features

- Dark theme with warm terracotta/coffee tones
- Custom animated cursor
- Auto-scrolling marquee banner
- Filterable menu cards (All / Drinks / Food / Desserts)
- Infinite-scroll reviews section
- Scroll-reveal animations
- Google Maps embed
- Fully mobile responsive
- Navbar hide/show on scroll

## 🗂️ Project Structure

```
atariyaa-cafe/
├── index.html        ← Main HTML file
├── css/
│   └── style.css     ← All styles
├── js/
│   └── main.js       ← All JavaScript
└── README.md         ← This file
```

## 🛠️ Technologies Used

- **HTML5** — Structure
- **CSS3** — Styling, animations, responsive layout
- **Vanilla JavaScript** — Interactions (no frameworks needed!)
- **Google Fonts** — Playfair Display, Space Grotesk, Bebas Neue

## 🚀 How to Run Locally

1. Download or clone this repo
2. Open `index.html` in any browser
3. That's it — no installation needed!

## 📦 How to Deploy (GitHub Pages)

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set branch to `main` → Save
4. Your site goes live at `https://YOUR-USERNAME.github.io/atariyaa-cafe`
